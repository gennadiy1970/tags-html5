Теги внедрения внешних ресурсов

iframe

Элемент <iframe> представляет собой вложенный контекст просмотра внешней страницы HTML внутри текущей страницы. Вы можете включить любое количество элементов <iframe> в документ, каждый из которых встраивает другой документ внутри <body> страницы. 

<iframe src="https://www.google.com/maps..." 
  width="960"
  height="450"
  frameborder="0"
  style="border:0"
  allowfullscreen>
</iframe>

Object

Элемент <object> позволяет отобразить внешний ресурс, который можно рассматривать как изображение или или ресурс, который должен обрабатывать плагин. 


<object data="../assets/html5-cheat-sheet.pdf"
    type="application/pdf"
    width="960"
    height="450">
</object>
